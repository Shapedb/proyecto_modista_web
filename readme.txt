/*Adaptacion de Pixeles*/
El siguiente comando crea que todos los "rem" dentro de CSS sean lo siguiente:

1rem = 10px

Este cambiara para adapatarse de mejor manera a diferentes pantallas.

Codigo de CSS:

html {
    font-size: 62.5%;
}
body {
    font-size: 16px;
}
